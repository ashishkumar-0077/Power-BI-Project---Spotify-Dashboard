# 📌 Project Overview

This Power BI dashboard provides an interactive analysis of Spotify music data. It helps users explore song performance, artist popularity, album distribution, monthly trends, and listening insights through dynamic visualizations.

---

# 🎯 Objectives

- Analyze Spotify music dataset
- Identify top-performing artists and songs
- Track monthly popularity trends
- Compare album types
- Analyze Explicit vs Non-Explicit songs
- Build an interactive Business Intelligence dashboard

# 📌 Key Insights

- Taylor Swift has the highest number of songs in the dataset.
- Album releases are nearly balanced across years.
- Singles contribute the largest share of songs.
- Monthly popularity remains relatively stable with seasonal variations.
- Explicit songs represent a significant portion of the dataset.
